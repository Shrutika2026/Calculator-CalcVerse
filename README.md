# 📖 Overview

CalcVerse is a complete calculation and conversion platform designed to provide everything from basic arithmetic to advanced academic, financial, scientific, and health-related calculations in one place.

The application features modern UI/UX, voice-enabled calculations, AI integration, calculation history management, and real-time conversion utilities.

# 🛠️ Installation Guide

## Prerequisites

Install:

* Node.js (v16 or higher)
* Git

---

## Clone Repository

```bash
git clone https://github.com/Shrutika2026/calc-verse.git
```

---

## Navigate to Project

```bash
cd calc-verse
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file if your currency converter uses an external API.

Example:

```env
VITE_CURRENCY_API_KEY=YOUR_API_KEY
```

---

## Run Development Server

```bash
npm run dev
```

Open:

```plaintext
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

The optimized production files will be generated inside:

```plaintext
dist/
```
